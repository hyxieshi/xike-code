# xike code

> 终端 AI 编程助手 — 类似 Claude Code / Open Code

**xike code** 是一个基于 TypeScript 实现的终端 AI 编程助手，旨在通过自然语言交互辅助开发者完成代码编写、重构、调试等日常开发任务。

## 特性

- 💬 自然语言驱动的终端交互
- 🛠 代码编写、重构与调试
- 🧠 多 Agent 协作
- 🔌 可扩展的 MCP 工具集成
- 📂 项目管理与代码理解

## 快速开始

```bash
# 安装
npm install -g xike-code

# 使用
xike
```
